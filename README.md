# 🎓 StudentHub Portal

A simple front-end web portal developed as a semester-long academic project. StudentHub provides a centralized platform where students, faculty, and administrators can access academic information such as courses, attendance, notices, and personal profiles.

---

## 📌 Project Overview

StudentHub is designed to simplify campus management by providing an easy-to-use interface for students and faculty. This project focuses on front-end development using HTML, CSS, and JavaScript.

---

## ✨ Features

- 🏠 Home Page
- 🔐 User Login
- 📝 Student Registration
- 📊 Dashboard
- 📚 Course Management
- ✅ Attendance Tracking
- 📢 Notices & Announcements
- 👤 Student Profile
- 📞 Contact Page
- ℹ️ About StudentHub

---

## 👥 User Roles

- **Student**
  - Register and Login
  - View Courses
  - Check Attendance
  - View Notices
  - Update Profile

- **Faculty**
  - Login
  - View Student Information
  - Manage Attendance
  - Post Notices

- **Administrator**
  - Manage Users
  - Manage Courses
  - Maintain Portal Information

## 🎯 Project Objective

The objective of StudentHub is to provide a centralized portal where students can manage academic activities such as viewing courses, tracking attendance, reading notices, and maintaining their profile in one place.

---

## 🚀 How to Run the Project

1. Download or clone this repository.
2. Open the project folder in Visual Studio Code.
3. Open `index.html` in your web browser.
4. Navigate through the portal using the navigation menu.
