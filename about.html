About StudentHub

StudentHub is a student-built web portal created as a semester-long project. Its goal is to give students and faculty a single place to manage everyday academic tasks such as viewing courses, checking attendance, and reading notices.

User Roles

Role Can do
Student Register, login, view courses, check attendance, view notices, edit profile
Faculty Login, post notices, mark attendance, view student list
Admin Manage users, manage courses, manage overall data

Project Goal

This project focuses on planning a clean, maintainable front-end structure before backend development begins.
